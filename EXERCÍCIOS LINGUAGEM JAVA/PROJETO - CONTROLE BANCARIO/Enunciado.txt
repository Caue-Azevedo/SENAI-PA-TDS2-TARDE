2)Crie uma classe com nome Cliente e atributos código e nome.

5)Crie uma classe CartaoCredito para modelar os objetos que  representarão os cartões de crédito. Considere a numeração e a  data como valores de texto.

6)Faça um teste criando objetos da classe CartaoCredito. Altere e imprima os atributos desses objetos.
